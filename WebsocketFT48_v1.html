WebSocket FT4/8 Help

Getting Started
Enter the SDRconnect server IP address (default 127.0.0.1) and port (default 5454), then click Connect. Once connected, all controls become active. Select a band and click the Decode button to begin decoding FT8 or FT4 signals. The Audio Level should be adjusted so that the Audio Meter is showing green bars to mid-scale and no orange or red bars.
Toolbar Buttons
Decode Off / OnStarts and stops the FT8/FT4 decoder. When active, the button turns green.
FT8 / FT4Switches between FT8 and FT4 decoding modes. The current mode is shown on the button.
Fast / Normal / DeepSets the decoder depth. Fast uses minimal CPU; Normal is the recommended default; Deep attempts more aggressive decoding at the cost of higher CPU usage.
Country Off / CQ / AllControls whether country names appear alongside decoded callsigns. CQ shows countries only on CQ messages; All shows them on every decode.
BandSelects the amateur radio band to tune to. Changing bands resets the decoder and retunes the SDR.
Hop Off / Hop OnOpens the Band Hopping modal where you can configure automatic cycling through multiple bands. When hopping is active, the Decode, Mode, and Band controls are locked.
MapOpens the Station Map showing all decoded stations plotted on a world map for the current session. Markers are colour-coded by band. Click any marker to see station details.
AGC Off / OnEnables or disables the Automatic Gain Control in SDRconnect. The recommended setting is disabled (AGC Off).
Clear (Decode Window)Clears the decoded messages list and map display.
Clear (Spectrum & Waterfall)Clears the spectrum and waterfall display.
Decode Window
Messages are displayed after each decode cycle. Left clicking on any message will open a split window showing all messages on that frequency. Right clicking on a message opens the sender's QRZ page in a new tab.
Station Map
Stations are plotted using their transmitted Maidenhead grid square when available, otherwise by country location. The map accumulates stations for the entire session and resets when you clear the decode panel, disconnect, or change bands. You can zoom in and out using the mouse wheel. To pan, left click and hold. Click any marker to see station details including callsign, Maidenhead locator or country, band, SNR, and the time last heard in UTC. A QRZ button appears in the popup for resolved callsigns, opening the station's QRZ page in a new tab.
Band Hopping
Configure up to 11 bands with individual cycle times and FT8/FT4 mode per band. The hopper cycles automatically, collecting decodes on each band for the configured time before moving to the next. Each band tracks its own decode, callsign, and country counts; use Reset Stats to clear them. Your band hopping configuration is saved automatically and restored the next time you open the app.
Spectrum & Waterfall
The spectrum display shows signal levels across the FT8/FT4 audio passband (0–3320 Hz). The waterfall below it shows signal history over time, with stronger signals appearing brighter.
Slow / FastSets the waterfall scroll speed. Slow is easier to read during active decoding; Fast shows more history.
CumulativeToggles cumulative frame averaging. When on, the spectrum averages multiple frames for a smoother display.
AutoscaleAutomatically adjusts the Min and Max dB levels to fit the current signal environment.
MinSets the minimum dB level of the colour scale. Signals below this level appear black. Adjust with the ▲▼ buttons or type a value directly.
MaxSets the maximum dB level of the colour scale. Signals above this level appear at full brightness. Adjust with the ▲▼ buttons or type a value directly.
ClearClears the spectrum and waterfall display.
Requirements
Requires SDRconnect running on your local network with WebSocket access enabled. For accurate FT8/FT4 decoding, your system clock must be synchronized to UTC within 1 second. Windows users should run a dedicated NTP sync tool such as Nettime or Dimension 4.
